# RPG Game

Текущий билд нерабочий — проводится рефакторинг.

## Изменения в рефакторинге
- изменение механики боя
- изменение структуры справочников

## Что сделано
- расчет количества урона
- универсальный расчет класса брони
- создан справочник со сложностью существ
- переработан справочник персонажа
- расчет шанса на попадание
- расчет модификатора характеристик

## Дальнейшие планы
- переработать основной цикл игры с учетом новых механик
- добавить уровень персонажа
- добавить новых существ
- добавить новые предметы (с параметром шанса получения % и типом объекта значений, например {'шкура': 1})
- добавить виды урона: колющий, дробящий, рубящий
- добавить виды состояний: отравление, воспаление, кровотечение
- добавить для существ уязвимость
- добавить для существ сопротивление

## Стек технологий
- Python 3.x
- PyTest
- Git
